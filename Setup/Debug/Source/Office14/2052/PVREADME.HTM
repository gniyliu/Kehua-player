关于 PowerPoint Viewer

使用 Microsoft PowerPoint Viewer，可查看在 Microsoft PowerPoint 2000、PowerPoint 2002、PowerPoint 2003、 PowerPoint 2007 和 PowerPoint 2010 中创建的演示文稿。PowerPoint Viewer 还能打开受密码保护的 PowerPoint 演示文稿。

多语言问题

PowerPoint Viewer 不支持 Microsoft Office 校对工具。

PowerPoint Viewer 不支持使用 PowerPoint 97 创建的双语演示文稿。

使用 PowerPoint Viewer

PowerPoint Viewer 不能与 64 位版的 Microsoft Office 并行安装。

PowerPoint Viewer 不支持打开受信息权限管理 (IRM) 保护的演示文稿。

PowerPoint Viewer 不支持运行宏、程序或者打开链接对象或嵌入对象。

将图片链接到演示文稿中时，PowerPoint Viewer 不支持显示以下图片格式： .eps、.pct、.emz、.cdr、.cgm、.wpg 和 .wmz。
